# 🚀 AI-Powered Food Menu Text Extraction  

**Revolutionizing menu digitization with AI-driven OCR, transforming unstructured food menu images into structured, actionable data.**  

## 🔹 Overview  
This project utilizes **OpenCV, PIL, Tesseract OCR, and EasyOCR** to extract text from food menu images with high precision. The extracted text is then **structured into JSON format** using an LLM, making it easy to integrate into various applications.  

## ⚡ Key Features  
✅ **Advanced Image Preprocessing:** Resizing, noise reduction, and binarization using OpenCV & PIL.  
✅ **Dual-OCR System:** Tesseract & EasyOCR for optimized text recognition.  
✅ **LLM-Powered Structuring:** Extracted text is intelligently formatted into JSON.  
✅ **Real-Time Processing:** Enables instant menu digitization for improved efficiency.  
✅ **Scalable & Efficient:** Designed for seamless integration into restaurant and food-tech applications.  

## 🛠 Tech Stack  
- **Programming Language:** Python  
- **Libraries:** OpenCV, PIL, Tesseract OCR, EasyOCR  
- **AI & NLP:** LLMs for text structuring
